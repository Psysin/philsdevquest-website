VS Code ist mein täglich genutzter Editor. Nach einer Weile merkt man: Die meisten nutzen nur 20% der Features.

## 1. Multi-Cursor

Halte `Alt` gedrückt und klicke an mehrere Stellen — du schreibst dann gleichzeitig an allen Positionen.

## 2. Integriertes Terminal

`Strg + J` öffnet das Terminal direkt in VS Code. Kein Fensterwechsel mehr nötig.

## 3. Format on Save

In den Einstellungen `editor.formatOnSave: true` aktivieren — Code wird beim Speichern automatisch formatiert.
