## Godot 4.6.1: Schnelles Update für mehr Stabilität

Nur drei Wochen nach dem Release von Godot 4.6 hat das Godot-Team bereits das erste Wartungs-Update veröffentlicht. Version 4.6.1 behebt insgesamt 38 bekannte Fehler und macht die Engine damit spürbar stabiler. Für alle, die gerade mit Godot einsteigen, ist das eine gute Nachricht: Die Grundlage, auf der ihr eure ersten Projekte baut, wird mit jedem solchen Update solider.

## Was wurde konkret behoben?

Besonders relevant für Einsteiger sind drei Bereiche, in denen das Update Verbesserungen bringt. Erstens wurden kritische Abstürze im Editor behoben, die bei bestimmten Aktionen auftreten konnten. Zweitens hat das Team Animationsfehler gefixt, die dazu führen konnten, dass Animationen nicht wie erwartet abgespielt wurden. Drittens wurden Probleme beim Erstellen von Skripten behoben, was gerade für Anfänger besonders ärgerlich sein kann, wenn der Editor beim Anlegen einer neuen Datei nicht mitmacht.

## Soll ich updaten? Und wenn ja, wie?

Das Godot-Team stuft das Update als sicher für ein Upgrade ein. Das bedeutet, ihr könnt eure bestehenden Projekte ohne grosse Risiken mit Godot 4.6.1 weiterführen. Dennoch empfiehlt das Team, vor dem Update ein Backup eurer Projektdateien anzulegen oder, falls ihr bereits mit Git arbeitet, sicherzustellen, dass alle Änderungen committet sind. Diese Vorsichtsmaßnahme ist generell eine gute Gewohnheit, egal wie klein ein Update ist.

## Wie bekomme ich das Update?

Das Update lässt sich direkt über die offizielle Godot-Website herunterladen. Wer den Godot-Launcher oder Steam nutzt, kann das Update dort ebenfalls beziehen. Die neue Version ersetzt einfach die alte Installation, bestehende Projekte werden automatisch erkannt. Wer gerade erst mit Godot startet, sollte direkt zur Version 4.6.1 greifen, da diese von Anfang an die stabilere Basis bietet.
