Programiz bietet einen kostenlosen Online-Compiler für Python direkt im Browser. Kein Setup, keine Installation — einfach Code eingeben und ausführen.

## Wann ist das nützlich?

- Du bist unterwegs und hast kein VS Code dabei
- Du möchtest schnell etwas ausprobieren, ohne ein neues Projekt anzulegen
- Du lernst gerade Python und willst Konzepte direkt ausprobieren
- Du möchtest jemandem Code zeigen, ohne Dateien schicken zu müssen

## Was Programiz noch bietet

Neben dem Compiler gibt es auf Programiz auch eine große Sammlung an Python-Tutorials und Beispielen — alles auf Englisch, aber gut verständlich für Einsteiger. Die Erklärungen sind knapp gehalten und praxisnah.

Für tieferes Lernen würde ich die offizielle Dokumentation empfehlen, aber als schnelles Werkzeug zwischendurch ist Programiz ideal.
