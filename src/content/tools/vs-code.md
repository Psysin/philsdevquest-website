VS Code ist mein Haupt-Editor für alles: Python-Skripte, dieses Astro-Projekt, Markdown-Dateien — alles läuft in VS Code. Er ist kostenlos, schnell und lässt sich durch Extensions beliebig erweitern.

## Extensions, die ich täglich nutze

**Für Python:**

- **Python** (Microsoft) — Syntax-Highlighting, IntelliSense, Debugging
- **Pylance** — bessere Autovervollständigung und Typ-Hinweise
- **Black Formatter** — automatische Code-Formatierung

**Für Astro:**

- **Astro** (offiziell) — Syntax-Support, IntelliSense für `.astro`-Dateien
- **Prettier** — automatische Formatierung beim Speichern

**Allgemein nützlich:**

- **GitLens** — Git-Historie direkt im Editor sehen
- **Error Lens** — Fehler werden direkt in der Zeile angezeigt, nicht nur in der Problemleiste

## Einstellungen, die ich empfehle

- `editor.formatOnSave: true` — Code wird beim Speichern automatisch formatiert
- `editor.wordWrap: on` — lange Zeilen umbrechen statt seitlich scrollen
- `files.autoSave: afterDelay` — automatisches Speichern alle paar Sekunden

Mit diesen Grundeinstellungen und den richtigen Extensions ist VS Code mehr IDE als simpler Texteditor.
